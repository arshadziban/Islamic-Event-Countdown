# Islamic Event Countdown

Islamic Event Countdown is a lightweight, responsive web application designed to present a real-time countdown to important Islamic dates. The project emphasizes a clean visual design, accessibility across device sizes, and straightforward deployment.

## Overview

The application currently highlights the upcoming first day of Ramadan and displays:

- Remaining days, hours, minutes, and seconds
- Event metadata (Gregorian and Hijri date context)
- A visual progress indicator toward the event

## Features

- Real-time countdown timer (auto-updating every second)
- Responsive layout for mobile, tablet, and desktop
- Elegant UI with Islamic-inspired visual styling
- Lightweight static architecture (no backend required)

## Technology Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Font Awesome (icons)
